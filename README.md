# CodeTrack-AI

A full-stack web application to help students plan and track coding and aptitude preparation.

## 🚀 Tech Stack

- Java
- Spring Boot
- MySQL
- React
- Git & GitHub

## 📌 Planned Features

- Coding Topic Tracker
- Aptitude Topic Tracker
- Daily Study Planner
- Smart Scheduler
- Progress Dashboard
- Revision Planner
- Pomodoro Timer
